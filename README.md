# demo-ios-weather
demo-ios-weather 是一个基于 Wilddog iOS SDK 开发的修改天气状态 demo，主要实现 Wilddog 数据多设备同步修改和加载。

### 本地运行

首先确认本机已经安装Xcode 运行环境，然后执行,模拟器或者真机：
```
git clone git@github.com:WildDogTeam/demo-ios-weather.git
cd  demo-ios-weather
```
